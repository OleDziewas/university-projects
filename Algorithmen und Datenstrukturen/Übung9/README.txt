Plot your data using Gnuplot:*
-Fill in your measurements in results.dat
-Run $ ./plot.plt results.dat > plot.jpeg 

*Gnuplot can be installed with $ sudo apt install gnuplot.

Alternatively, plot your data using LaTeX:**
-Fill in your measurements in results.dat
-Run $ pdflatex ./plot.tex

**LaTeX can be installed with $ sudo apt install texlive-latex-extra.
