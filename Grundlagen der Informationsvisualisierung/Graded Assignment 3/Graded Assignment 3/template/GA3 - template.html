Explanations:

Task (1)

  • The node size encodes the number of medals won by each country.
  • The size of the categories is encoded by the number of participants.
  • The thicknis of a link depicts how many unique athletes of a country have won medals in a specific category.
Task (2)
  • For better readability the smaller subgroup (categories) of the bipartite graph was put in the center. The larger subgroup (countries) was put around it with a certain distance to have less overlapping.
  • Instead of removing nodes and links you can click on a node to get only the relevant links to that node. By clicking elsewhere or again on that node it goes back to the original view.
